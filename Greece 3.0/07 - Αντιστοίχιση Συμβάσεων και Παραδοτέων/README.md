# Μητρώο Αντιστοίχισης Συμβάσεων και Παραδοτέων

Το μητρώο αυτό ξεκινά από τα συμβατικά αντικείμενα που εμφανίζονται στον φάκελο [03 - Συμβάσεις Έργων Φυσικών Προσώπων](../03%20-%20Συμβάσεις%20Έργων%20Φυσικών%20Προσώπων/).

Σε αυτή την πρώτη έκδοση, η αντιστοίχιση γίνεται ανά πρόσωπο και κύριο συμβατικό αντικείμενο. Όπου δεν έχει εντοπιστεί συγκεκριμένο αρχείο παραδοτέου, η κατάσταση παραμένει `Προς αντιστοίχιση` ή `Μη εντοπισμένο`.

## Σημειώσεις

- Η ένδειξη `Μη εντοπισμένο` δεν αποδεικνύει από μόνη της ότι το παραδοτέο δεν υπάρχει. Σημαίνει ότι δεν έχει εντοπιστεί στα διαθέσιμα αρχεία του project.
- Για να θεωρηθεί κάτι `Αντιστοιχισμένο`, πρέπει να υπάρχει συγκεκριμένο αρχείο, URL, παραλαβή, έκθεση, τεχνική τεκμηρίωση ή άλλο ελέγξιμο τεκμήριο.
- Το μητρώο πρέπει να ενημερώνεται κάθε φορά που προστίθεται νέο PDF, νέα σύμβαση, παραλαβή, τιμολόγιο ή τεχνικό παραδοτέο.
