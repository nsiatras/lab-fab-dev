# Απευθείας Αναθέσεις

Ο φάκελος συγκεντρώνει αποφάσεις απευθείας ανάθεσης προμηθειών ή παροχής υπηρεσιών που συνδέονται με το έργο **«Μεταφραστική Έρευνα σε τεχνολογίες βιώσιμων υλικών» / ΟΠΣ ΤΑ 5179171 / Ε-12575**.

## Σημείωση

Οι απευθείας αναθέσεις αυτού του φακέλου διαχωρίζονται από τις συμβάσεις έργου φυσικών προσώπων, επειδή αφορούν αποφάσεις προμήθειας ή παροχής υπηρεσίας και όχι ατομικές συμβάσεις έργου.

Η υπόθεση του **MIT Center for Bits and Atoms** τεκμηριώνεται σε ξεχωριστό υποφάκελο, επειδή περιλαμβάνει περισσότερα από ένα έγγραφα: αίτηση προμήθειας, πρόσκληση, απόφαση, συγκρότηση επιτροπών, οριστική κατακύρωση και εντολή πληρωμής.
