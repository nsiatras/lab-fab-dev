Στις προκηρύξεις του φακέλου εμφανίζεται η ακόλουθη ουσιαστική διατύπωση:

> μέσω σύναψης συμβάσεων έργου, στις οποίες θα προσδιορίζεται η αποκλειστική απασχόληση στην Πράξη.

> μέσω σύναψης συμβάσεων ανάθεσης έργου, στην οποία/στις οποίες θα προσδιορίζεται η αποκλειστική απασχόληση στην Πράξη.

## Σημειώσεις

- Σε όλες τις προκηρύξεις που ελέγχθηκαν στον φάκελο υπάρχει ρητή αναφορά ότι οι συμβάσεις θα προσδιορίζουν την **αποκλειστική απασχόληση στην Πράξη**.
- Ο **Κωνσταντίνος Νικητίδης** εμφανίζεται να συμμετείχε ταυτόχρονα σε δύο διαφορετικά έργα, των οποίων οι προκηρύξεις ανέφεραν ότι οι συμβάσεις αφορούν **αποκλειστική απασχόληση στην Πράξη**.
- Στο έργο **Μεταφραστική Έρευνα σε Τεχνολογίες Βιώσιμων Υλικών**, `Ε-12575 / ΟΠΣ ΤΑ 5179171`, εμφανίζεται σύμβαση του Κωνσταντίνου Νικητίδη για την περίοδο **23/01/2023 - 22/01/2024**.
- Στο έργο **Δίκτυο Ενδυνάμωσης Μεταφοράς Τεχνολογίας**, `Ε-12545 / ΟΠΣ ΤΑ 5155000`, εμφανίζονται συμβάσεις του Κωνσταντίνου Νικητίδη για τις περιόδους **21/03/2023 - 20/09/2023** και **21/09/2023 - 31/12/2023**.
- Επομένως, από τα τοπικά αρχεία προκύπτει χρονική επικάλυψη από **21/03/2023 έως 31/12/2023**.
- Η παραπάνω επικάλυψη καταγράφεται ως σημείο ελέγχου, επειδή αφορά δύο έργα με προκηρύξεις που αναφέρουν ρητά την **αποκλειστική απασχόληση στην Πράξη**.
