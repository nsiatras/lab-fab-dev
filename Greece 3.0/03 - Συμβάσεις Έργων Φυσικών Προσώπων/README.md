# Συμβάσεις Έργων Φυσικών Προσώπων

Αυτός ο φάκελος συγκεντρώνει συμβάσεις έργου φυσικών προσώπων, κυρίως για το έργο **«Μεταφραστική Έρευνα σε τεχνολογίες βιώσιμων υλικών»** με κωδικό **ΟΠΣ ΤΑ 5179171 / Ε-12575**.

Τα αρχεία είναι οργανωμένα ανά φυσικό πρόσωπο και περιλαμβάνουν ημερομηνίες σύμβασης, συνολική αμοιβή, σύνδεσμο Διαύγειας και αντικείμενο σύμβασης.

## Σύνολα ανά πρόσωπο

### Παρατήρηση για χρονική επικάλυψη συμβάσεων

Στο αρχείο του Κωνσταντίνου Νικητίδη καταγράφεται χρονική επικάλυψη συμβάσεων μεταξύ διαφορετικών έργων. Στο έργο **Ε-12575 / ΟΠΣ ΤΑ 5179171** εμφανίζεται σύμβαση για το διάστημα **2023-01-23 έως 2024-01-22**, ενώ στο έργο **Δίκτυο Ενδυνάμωσης Μεταφοράς Τεχνολογίας / Ε-12545** εμφανίζονται συμβάσεις για τα διαστήματα **2023-03-21 έως 2023-09-20** και **2023-09-21 έως 2023-12-31**. Αυτό σημαίνει ότι από **2023-03-21 έως 2023-12-31** ο Κωνσταντίνος Νικητίδης εμφανίζεται ταυτόχρονα σε δύο διαφορετικά έργα, παρότι οι σχετικές προκηρύξεις αναφέρουν απασχόληση για την **«αποκλειστική απασχόληση στην πράξη»**.

## Πρόσωπα με ποσά από άλλα έργα

Ορισμένα αρχεία περιλαμβάνουν και συμβάσεις ή αμοιβές εκτός του Ε-12575:

- **Χριστόφορος Ρεκατσίνας**: περιλαμβάνει επίσης ποσά για τα έργα `Reinforcing the AI4EU Platform by Advancing Earth Observation Intelligence, Innovation and Adoption` και `AI4TRUST`.
- **Κωνσταντίνος Νικητίδης**: περιλαμβάνει επίσης ποσά για το έργο `Δίκτυο Ενδυνάμωσης Μεταφοράς Τεχνολογίας` και αναφορά πληρωμής για συμμετοχή σε πακέτα εργασίας `WP1`, `WP4`, `WP7`.
- **Φίλιππος Θ. Τουρλομούσης**: περιλαμβάνει επίσης ποσά για το έργο `Theseus - A First of a kind Hub for circularity demonstrator for Attica and peripheral regions`, καθώς και κατ' αποκοπή αμοιβή 24.193,55 ευρώ πλέον ΦΠΑ.
- **Αντονέλλα Ροζάρια Νεφέλη Ποντίλλο**: περιλαμβάνει επίσης ποσό για το έργο `Theseus-A First of a kind Hub for circularity demonstrator for Attica and peripheral regions`.

## Σημειώσεις υπολογισμού

- Η στήλη **Σύνολο Ε-12575** περιλαμβάνει μόνο εγγραφές που ανήκουν στο έργο **ΟΠΣ ΤΑ 5179171 / Ε-12575**.
- Η στήλη **Σύνολο όλων των έργων** περιλαμβάνει όλες τις εγγραφές πινάκων που υπάρχουν στο αντίστοιχο αρχείο.
- Η κατ' αποκοπή αμοιβή του Φίλιππου Θ. Τουρλομούση αναφέρεται δύο φορές στο ίδιο αρχείο, στον τίτλο της ενότητας και στο σώμα του κειμένου. Στα σύνολα μετρήθηκε μία φορά.
- Το αρχείο `2026-01-12-συμβαση_έργου_τουρλομουση.md` τεκμηριώνει τη σύμβαση του έργου `Theseus` που ήδη εμφανίζεται και στο συγκεντρωτικό αρχείο `filippos_tourlomousis-contracts.md`. Δεν προσμετράται δεύτερη φορά στα σύνολα.
- Στο αρχείο `stavros_tzikas-contracts.md`, η εγγραφή της 01/06/2026 εμφανίζει το ποσό ως `3.385,20,80 €`. Στους υπολογισμούς του README λογίζεται ως `3.385,20 €`.
- Η κατ' αποκοπή αμοιβή αναφέρεται ως **24.193,55 ευρώ πλέον ΦΠΑ**. Το τελικό σύνολο τη συμπεριλαμβάνει χωρίς ΦΠΑ, επειδή έτσι εμφανίζεται στο αρχείο. Με ΦΠΑ 24%, η συγκεκριμένη αμοιβή αντιστοιχεί πρακτικά σε 30.000,00 ευρώ.
- Το README είναι συνοπτικό ευρετήριο και πίνακας υπολογισμών με βάση τα τοπικά αρχεία του φακέλου. Για πλήρη τεκμηρίωση πρέπει να ανατρέχει κανείς στα επιμέρους markdown αρχεία και στους συνδέσμους Διαύγειας.
