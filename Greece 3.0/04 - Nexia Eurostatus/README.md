# Nexia Eurostatus

Αυτός ο φάκελος συγκεντρώνει έγγραφα και δεδομένα που αφορούν τη Nexia Eurostatus A.E. στο πλαίσιο του έργου Ε-12575 / ΟΠΣ ΤΑ 5179171 με τίτλο «Μεταφραστική Έρευνα σε τεχνολογίες βιώσιμων υλικών».

Το υλικό αφορά κυρίως:

- σύμβαση και τροποποιήσεις για υπηρεσίες ανεξάρτητου ορκωτού ελεγκτή,
- αποφάσεις σχετικά με παραλαβή εκθέσεων ελέγχου,
- μη επιλέξιμες ή προσωρινά μη συμπεριλαμβανόμενες δαπάνες,
- πίνακα χρηματικών ενταλμάτων από τη Διαύγεια.
